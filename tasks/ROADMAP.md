# Roadmap

## Roadmap Purpose

This roadmap organizes delivery for an Arabic-first reusable Restaurant CMS Platform. The first implementation is a seafood restaurant.

## Phase 0: Documentation And Decisions

- Confirm Arabic-first product direction.
- Confirm `/ar` as the default public route.
- Confirm `/en` as the optional English public route.
- Confirm RTL for Arabic and LTR for English.
- Confirm admin dashboard is Arabic-first.
- Confirm every editable public field supports Arabic and English.
- Confirm seafood menu categories and pricing rules.
- Confirm no permanent restaurant text is hardcoded in application code.

## Phase 1: Platform Foundations
