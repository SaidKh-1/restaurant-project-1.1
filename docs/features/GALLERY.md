# Gallery

## Purpose

The gallery showcases seafood dishes, fresh fish displays, restaurant atmosphere, kitchen quality, events, and customer-facing brand trust.

## Image Fields

Each gallery image should support:

- Image asset.
- Arabic caption.
- English caption.
- Arabic alt text.
- English alt text.
- Category.
- Sort order.
- Featured state.
- Active state.

## Gallery Categories

Suggested configurable categories:

- الأسماك الطازجة
- المأكولات البحرية
- الأطباق الخاصة
- المطعم
- الفعاليات

Categories should be editable, not hardcoded.

## Admin Actions

Admin can:

- Upload images.
- Edit captions.
- Edit alt text.
- Reorder images.
- Feature images.
- Hide images.
- Delete or archive images.
- Use images in homepage, gallery, hero, offers, menu, manager profile, or SEO where appropriate.

## Public Display Rules

- Arabic gallery content is primary.
- English gallery content is optional.
- Images should be optimized for mobile.
- Missing captions should not break public layout.
- Gallery should not expose unpublished images.

## Accessibility And SEO

- Alt text is required for important public images.
- Captions should support Arabic and English.
- Open Graph images should come from approved media.
- Large gallery images should not harm performance.

## Future Enhancements

- Media library reuse.
- Bulk upload.
- Image cropping.
- CDN transformations.
- Event albums.
- Instagram integration.
