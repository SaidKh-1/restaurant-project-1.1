# Home Page Builder

## Purpose

The homepage builder controls the Arabic-first public homepage for the seafood restaurant. The owner should manage homepage content and visibility from the dashboard without code changes.

## Arabic-First Homepage Sections

- الرئيسية
- منيو اليوم
- الأسماك الطازجة
- المأكولات البحرية
- الأطباق الخاصة
- طرق الطهي
- العروض
- عن المطعم
- مدير المطعم
- معرض الصور
- التقييمات
- الحجز
- تواصل معنا
- الخريطة
- الفوتر

## Section Controls

Admin can:

- Show or hide homepage sections.
- Edit Arabic and English titles.
- Edit Arabic and English descriptions.
- Choose featured menu items.
- Choose active offers.
- Choose gallery highlights.
- Choose featured reviews.
- Manage manager section content through the manager profile.
- Manage hero images through the hero slider.

Admin can reorder homepage sections later if possible.

## Hero Slider

Hero slides should support:

- Hero image.
- Arabic title.
- English title.
- Arabic subtitle.
- English subtitle.
- Button labels.
- Button links.
- Visibility.
- Sort order.

Hero images are editable from admin.

## Seafood Homepage Needs

The homepage should prioritize:

- Fresh fish availability.
- Seafood menu discovery.
- Cooking methods.
- Special dishes.
- Offers.
- Reservation and WhatsApp actions.
- Manager or owner trust section.
- Map and contact details.

## Content Source Rule

Homepage text should come from admin-managed content. No permanent seafood restaurant marketing copy should be hardcoded in application code.
