# Offers

## Purpose

Offers let the seafood restaurant promote discounts, daily deals, bundles, seasonal seafood specials, and limited-time campaigns.

## Content Fields

Each offer should support:

- Arabic title.
- English title.
- Arabic description.
- English description.
- Image.
- Arabic and English alt text.
- Start date.
- End date.
- Active state.
- Featured state.
- Optional link or call to action.

## Arabic-First Public Display

Offers may appear in:

- العروض section.
- Homepage.
- Menu-related sections.
- WhatsApp or reservation call-to-action areas.

Arabic content is primary. English content appears under `/en` when available.

## Admin Actions

Admin can:

- Create offers.
- Edit offers.
- Publish or unpublish offers.
- Feature offers.
- Archive expired offers.
- Upload offer images.
- Manage Arabic and English fields.

## SEO Rules

- Arabic offer metadata is primary.
- English metadata is supported.
- Offer Open Graph images should be configurable.
- Expired offers should not mislead customers or search engines.

## Future Enhancements

- Scheduled offers.
- Coupon codes.
- Menu item associations.
- Offer analytics.
- WhatsApp campaign links.
